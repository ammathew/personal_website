Projects

CNKT is a full app that I programmed in 2015. It is a twitter platform that allows users to find people who are tweeting about the same things as they are (the user highlights key words or phrases from their own twitter timeline, and the app retrieves other tweets containing the same key words or phrases). It provides a dashboard where users can easily interact with these other users and their tweets. It takes payments via Stripe. The app has not been actively maintained since 2014, and so works fully only in Firefox and older versions of Chrome.

Live Website: http://cnkt-dev.sunship.co/
